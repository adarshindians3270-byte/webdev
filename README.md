# Frontend Learning Projects

This repository contains small frontend projects built during my learning phase.
The focus is on understanding **HTML structure, JavaScript logic, and DOM behavior**
using **Vanilla JavaScript**.

Projects here are created to explore concepts step-by-step rather than to build
fully polished products.

## Purpose
- Learn core web fundamentals
- Practice JavaScript logic and DOM manipulation
- Understand browser APIs and event handling
- Build functional UI behavior without frameworks

## Tech Stack
- HTML5
- Vanilla JavaScript (ES6+)
- CSS (manual in some projects, template / AI-assisted in others)
